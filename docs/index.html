AutoToc

Table of contents generator

About

AutoToc generates a table of contents list based on headings within an HTML document.

    Example

    Add a list element to your page:

    <ol id="toc">
</ol>

    Add AutoToc.js file to the page and run the AutoToc function, providing the list element id as an parameter:

    
<script src="AutoToc.js"></script>
<script>
  AutoToc("toc");
</script>
    Download AutoToc.js View on Github

    Options

    Scope

    By default, AutoToc will provide a table of contents based on all headings available within the body html element. You can limit the table of contents scope by providing the id of a container element as a secondary, optional parameter.

    <script src="AutoToc.js"></script>
<script>
  AutoToc("toc", "contents");
</script>

    Default options

    There are additional options available via the AutoToc.defaultOptions object.

    AutoToc.defaultOptions = {
  startLevel: 1,
  depth: 3,
  showTopLinks: false,
  topLinkToParentToc: false,
  topLinkClass: "autoTocTopLink",
  topBodyId: "top",
};

    Usage:

    <script src="AutoToc.js"></script>
<script>
  AutoToc("toc", "contents", { startLevel: 2, showTopLinks: true });
</script>
